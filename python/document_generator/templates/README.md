# Шаблоны документов

Здесь хранятся шаблоны для генерации документов и коммерческих предложений.

## Формат шаблонов

Шаблоны используют синтаксис Jinja2 для подстановки переменных:

- `{{ variable_name }}` - подстановка переменной
- `{% if condition %}...{% endif %}` - условные блоки
- `{% for item in list %}...{% endfor %}` - циклы

## Пример данных для шаблона

```python
{
    "company_name": "ООО Рога и Копыта",
    "client_name": "Иван Иванов",
    "date": "2024-01-15",
    "items": [
        {"name": "Услуга 1", "price": 10000},
        {"name": "Услуга 2", "price": 20000}
    ],
    "total": 30000
}
```
