# 🚀 Quick Start - SimpleDoc Block Editor

## Что это?

**SimpleDoc v1.0** - визуальный блочный редактор документов с реальной страницей A4, drag-and-drop блоками и точным экспортом в PDF.

---

## ⚡ Быстрый старт

### 1. Установка зависимостей

```bash
cd simpledoc
npm install
```

### 2. Запуск приложения

```bash
npm run tauri dev
```

Приложение откроется автоматически! 🎉

## 🎯 Основные возможности

### ✅ Создание блоков
1. Кликните на блок в **Block Library** (слева)
2. Блок появится на странице
3. Перетащите его в нужное место

### ✅ Редактирование
- **Single click** - выбрать блок
- **Double click** - редактировать текст inline
- **Drag** - переместить блок
- **Drag handles** - изменить размер
- **Properties Panel** (справа) - настроить стили

### ✅ Выделение
- **Click** - выбрать один блок
- **Ctrl+Click** - добавить к выделению
- **Ctrl+A** - выбрать все
- **Escape** - снять выделение

### ✅ Навигация
- **Zoom**: `Ctrl+Wheel` или `Ctrl+/-/0`
- **Pan**: Middle mouse или `Space+Drag`
- **Grid**: `Ctrl+G` для включения/выключения

## ⌨️ Горячие клавиши

### Файл
- `Ctrl+N` - Новый документ
- `Ctrl+S` - Сохранить
- `Ctrl+E` - Экспорт

### Редактирование
- `Ctrl+C` - Копировать
- `Ctrl+V` - Вставить
- `Ctrl+D` - Дублировать
- `Delete` - Удалить блок
- `Escape` - Снять выделение

### Вид
- `Ctrl+0` - Сброс zoom
- `Ctrl+=` - Zoom in
- `Ctrl+-` - Zoom out
- `Ctrl+G` - Сетка вкл/выкл

### Панели
- `Ctrl+B` - Показать/скрыть Block Library
- `Ctrl+P` - Показать/скрыть Properties

## 📦 Типы блоков

### Текстовые
- **Heading 1** - Крупный заголовок (32px, bold)
- **Heading 2** - Средний заголовок (24px, semibold)
- **Heading 3** - Малый заголовок (20px, semibold)
- **Paragraph** - Параграф текста (14px)
- **Text** - Базовый текст (16px)

### Медиа
- **Image** - Изображения с фильтрами
- **Table** - Таблицы (по умолчанию 3×3)

### Утилиты
- **Spacer** - Вертикальный отступ

---

## 🎨 Редактирование блоков

### Position & Size
В **Properties Panel** (справа):
- **X, Y** - Позиция на странице
- **Width, Height** - Размер блока

### Text Styles
Для текстовых блоков:
- **Font Size** - Размер шрифта (8-200px)
- **Font Weight** - Толщина (300-700)
- **Alignment** - Выравнивание (left/center/right/justify)
- **Color** - Цвет текста

### Layer Controls
- **⬆️ Forward** - Переместить вперёд
- **⬇️ Backward** - Переместить назад
- **🔒 Lock** - Заблокировать блок

## 🐛 Если что-то не работает

### Проблема: Не запускается Tauri
```bash
# Проверьте зависимости Tauri
cargo --version
rustc --version

# Установите Tauri prerequisites:
# https://tauri.app/v1/guides/getting-started/prerequisites
```

### Проблема: Ошибки TypeScript
```bash
# Запустите type checking
npm run check
```

### Проблема: Не видно блоков
- Проверьте, что документ создан (нажмите `Ctrl+N`)
- Убедитесь, что панели открыты (`Ctrl+B` и `Ctrl+P`)
- Попробуйте сбросить zoom (`Ctrl+0`)
